analisador_dados.analisar_dados_basico(df)[source]

Realiza uma análise básica do DataFrame fornecido.

Exibe a quantidade total de dados, a distribuição por gênero e a quantidade de registros sem informação sobre a educação dos pais.

Parameters:

df (pandas.DataFrame) – O DataFrame a ser analisado.

analisador_dados.carregar_dados()[source]

Carrega dados de um arquivo CSV ou JSON para um DataFrame do Pandas.

Pergunta ao usuário o caminho do arquivo e tenta carregá-lo. Suporta arquivos nos formatos CSV e JSON.

Returns:

Um DataFrame contendo os dados carregados, ou None se ocorrer algum erro.

Return type:

pandas.DataFrame or None

analisador_dados.consultar_dados_coluna(df)[source]

Permite ao usuário consultar estatísticas de uma coluna numérica específica do DataFrame selecionando por número. Reexibe as opções em caso de erro.

analisador_dados.limpar_dados(df)[source]

Realiza a limpeza dos dados do DataFrame.

Remove registros com valores nulos na coluna ‘Parent_Education_Level’ e preenche valores nulos na coluna ‘Attendance (%)’ com a mediana.

Parameters:

df (pandas.DataFrame) – O DataFrame a ser limpo.

Returns:

O DataFrame com os dados limpos, ou None se nenhum dado foi carregado.

pandas.DataFrame or None
